# Security Policy

## Reporting a vulnerability

Do not report security vulnerabilities through public GitHub Issues, Discussions, pull requests, screenshots, or trip-log attachments.

Email **admin@jarsoft.co.nz** with:

- the affected EcoDrive version;
- the affected feature or service;
- clear reproduction steps;
- the likely impact; and
- any suggested mitigation.

Do not include live credentials, access tokens, Google Play purchase tokens, private contact data, unredacted trip logs, exact home addresses, registration plates, VINs, or other personal information unless Jarsoft specifically requests it through a secure channel.

Jarsoft Limited will acknowledge a good-faith report, investigate it, and coordinate remediation and disclosure where appropriate. Please avoid accessing, modifying, retaining, or disclosing data that is not your own, and do not disrupt production services.

## Supported versions

Security fixes are provided for the current Google Play release and any newer test release that Jarsoft expressly identifies as supported.
