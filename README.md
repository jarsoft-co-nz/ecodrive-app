# EcoDrive public information

This repository contains public information for **EcoDrive**, an Android driving, navigation, trip-recording, OBD-II, and efficiency-coaching application operated by **Jarsoft Limited**.

## Privacy summary

EcoDrive is designed as a local-first application. Trip histories, GPS samples, OBD-II readings, vehicle profiles, and most settings are stored on the user's device rather than automatically uploaded to Jarsoft.

Online features transmit only the information needed for the requested service. This can include precise route coordinates sent to Jarsoft routing services, destination text or coordinates sent to a geocoding provider, and pseudonymous subscription information processed through Google Play and Jarsoft's entitlement service.

The current application does not include advertising, behavioural analytics, or third-party crash-reporting SDKs.

See the [Privacy Policy](./privacy-policy.md) for complete details.
